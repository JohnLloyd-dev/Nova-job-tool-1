The Role
We are building a next-generation payment orchestration platform that will revolutionize how merchants integrate and optimize multiple payment providers globally. We are looking for an experienced Senior Tech Lead – Backend to lead technical development, manage a backend engineering team, and own the DevOps infrastructure.
This is a hands-on leadership role where you will spend significant time writing production code while providing technical direction and mentorship.

Responsibilities
Hands-On Development

Write production-grade code daily, delivering features end-to-end from database design to API implementation and deployment
Build backend services and RESTful APIs supporting business logic, transaction processing, and merchant integrations
Integrate multiple payment providers and financial services using unified interfaces and adapter patterns
Design data models and algorithms to support high-volume, low-latency transaction processing
Debug, troubleshoot, and optimize system performance in production environments
Technical Leadership
Lead and manage a team of backend engineers, including task assignment, sprint planning, and performance management
Own backend architecture and key design decisions for a microservices-based platform
Conduct code reviews and mentor engineers through pair programming and technical guidance
Collaborate with Product to define requirements, estimate timelines, and establish engineering best practices
DevOps & Infrastructure
Own and manage CI/CD pipelines, container orchestration, and cloud infrastructure
Implement Infrastructure as Code (IaC) and manage cloud environments (AWS, GCP, or Azure)
Set up and maintain databases, caching layers, monitoring, and observability tools
Ensure secure, zero-downtime deployments, rollback strategies, and production stability
